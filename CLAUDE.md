## Project Overview

pdfstrip is a Python CLI tool that removes passwords from PDF files. Uses Python 3.13 and is managed with `uv`.

## Commands

- **Run**: `uv run pdfstrip <files> -p <password>`
- **Help**: `uv run pdfstrip --help`
- **Tests**: `uv run pytest`
- **Add dependency**: `uv add <package>`
- **Sync environment**: `uv sync`
